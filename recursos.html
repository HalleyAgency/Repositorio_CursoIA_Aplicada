ChatGPT (OpenAI)

El popular modelo que masificó la IA conversacional, muy versátil para una amplia gama de tareas. Visitar

Claude (Anthropic)

Conocido por su gran ‘ventana de contexto’, ideal para analizar documentos extensos y mantener conversaciones largas. Visitar

Generación de Contenido Visual y Presentaciones

Midjourney

Uno de los generadores de imágenes de mayor calidad y más artísticos, accesible a través de Discord. Visitar

DALL-E 3 (vía Microsoft Copilot)

El modelo de OpenAI, excelente en seguir prompts complejos y generar texto dentro de las imágenes. Probar

Gamma.app

La herramienta de IA para generar presentaciones y documentos completos a partir de un simple prompt. Visitar

Plataformas de Automatización

n8n.io

Plataforma de automatización ‘source-available’. Ideal para ingenieros por su flexibilidad y potencia. Visitar

Make (formerly Integromat)

Una potente plataforma visual para la creación de flujos de trabajo complejos sin necesidad de código. Visitar

Zapier

El líder del mercado en automatización no-code, conocido por su facilidad de uso y su enorme cantidad de aplicaciones integradas. Visitar

Editores de Código y Herramientas de Desarrollo

Visual Studio Code

El editor de código más popular del mundo. Instálalo para poder usar extensiones de IA. Descargar

GitHub Copilot

La extensión de autocompletado de código por IA más famosa. Se integra directamente en VS Code. Visitar

Cursor

Un editor de código ‘IA-first’ que integra funcionalidades de chat y edición de código de forma nativa. Visitar
